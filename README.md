# LOGCOM prototype

Nowoczesny prototyp webowy aplikacji LOGCOM v0.3 dla dowódcy kompanii logistycznej.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Zustand
- lucide-react
- Recharts
- Framer Motion

## Uruchomienie

```bash
npm install
npm run dev
```

## Dostępne skrypty

- `npm run dev` - lokalny serwer Vite
- `npm run build` - kompilacja TypeScript i build produkcyjny
- `npm run lint` - statyczna analiza ESLint
- `npm run preview` - podgląd buildu
